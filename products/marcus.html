Pull-Based Architecture

No idle agents, no complex orchestration. Agents request tasks when ready. Marcus intelligently assigns based on skills and context.

# How agents work autonomously
while project.has_tasks():
    task = agent.request_next_task()
    result = agent.complete_task(task)
    agent.report_progress(result)
