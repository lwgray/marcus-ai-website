Read Documentation View Source

Comprehensive Monitoring

Service Discovery

Automatically discovers running Marcus instances through file-based service registry. No configuration needed.

# Start with auto-discovery
seneca start

# Or specify Marcus instance
seneca start --marcus-server /path/to/marcus

Real-time Visualization

Monitor agent conversations, task progress, and system events as they happen through WebSocket connections.

# Check system status
seneca status

# View recent logs
seneca logs --tail 100

Interactive Canvas

Vue Flow-based canvas for visualizing agent relationships, workflow dependencies, and communication patterns.

# Components available
- WorkflowCanvas
- AgentNodes
- DecisionNodes
- KnowledgeGraph
